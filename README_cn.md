## 跨云CROSSIM解决方案

跨云CROSSIM是一款采用了“跨生态共享”技术的即时通信解决方案。

主要特性有：
（1）跨界单聊
（2）跨界群聊
（3）跨界小程序

主要包括一下项目：

| [GitHub仓库地址(主站)](https://github.com/openbpi/crossim)      | [码云仓库地址(镜像)](https://gitee.com/apowner)        | 说明                                                                                      | 备注                                           |

| [wx-chat](https://github.com/wildfirechat/wx-chat)           | [wx-chat](https://gitee.comchat/wx-chat)           | 微信小程序平台的Demo                                                                      |                                                |
